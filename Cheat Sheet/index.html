CSS Selectors and Specificity

Selector Type Sepcificity HTML Example CSS Example
ID Most specific. <div id="identified">This div has a special ID on it!</div> #identified { background-color: skyblue;}
Class Second-most specific. <p class="red">This paragraph has red text.</p> .red { color: #f33;}
Type Least specific. <span>Here's a span with some text.</span> span { background-color: skyblue;}
